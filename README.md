# Income and Bike Sales Analysis

## Project Description
A data analysis and visualization project focused on analyzing customer income patterns and bike purchasing behavior using Excel dashboards, pivot tables, and business analytics techniques.

This project explores how factors such as income, gender, age, occupation, and demographics influence bike purchase decisions. The analysis was performed using Excel’s analytical tools including pivot tables, filtering, sorting, and dashboard visualizations to generate meaningful business insights.

The goal of this project is to practice real-world data analysis, reporting, dashboard creation, and business intelligence concepts commonly used in data analytics and data science workflows.
